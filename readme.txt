Actualiza twitter usando niq.mx

== Description ==

Actualiza twitter usando niq.mx(apoyando al acortador mexicano). Es una pequeña adaptación del plugin Twitter updater w/ TinyURL de Victoria Chan y Jonathan Dingman.

== Installation ==


1. Upload `techniq2twitter.zip` al directorio `/wp-content/plugins/`
2. Activa el plugin desde el menu `Plugins` en WordPress
3. Ir a  Tools -> techniq2twitter, escribe el texto de los enlaces automáticos y los datos de tu cuenta de twitter
4. Listo, ya quedó.
